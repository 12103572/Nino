比较运算符
    运算符：
      <        小于
      <=       小于等于
      >        大于
      >=       大于等于
      ==       等于
      !=       不等于

    语法：
        左表达式　<　右表达式
    说明：
        比较运算符返回布尔类型的值
    示例：
        100 < 200   #  结果为True
        100 <= 200 # 结果为True
比较运算的数学表达方式
    x = 61
    0 <= x <= 100

bool　运算
    运算符
        not   and  or
布尔非操作　not
    语法：
        not x
        注：x代表表达式
    作用：
        对x进行布尔取非，bool(x)为True则返回False，否则返回True
    示例：
        not True 则返回为　False
        not False 则返回为　True
布尔与操作　and
    语法：
        x and y
        注：x,y代表表达式
    作用：
        优先返回假值对象
        当x的布尔值为False，返回x，否则返回y
    示例：
        True  and True      # True
        False and True      # False
        True  and False     # False
        False and False     # False
        100   and 3.14      # 3.14
        3.14  and 100       # 100
        0     and 3.14      # 0
        3.14  and 0         # 0

布尔或操作　or
    语法：
        x or y
    作用：
        优先返回真值对象
        当x为True时，返回x，否则返回y
    示例：
        True  or  True      # True
        False or  True      # True
        True  or  False     # True
        False or  False     # False
        100   or  3.14      # 100
        3.14  or  100       # 3.14
        0     or  3.14      # 3.14
        3.14  or  0         # 3.14
